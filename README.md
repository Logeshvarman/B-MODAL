# B-MODEL

## Run Locally

Clone the project

```bash
  git clone https://github.com/Logeshvarman/B-MODAL
```

Go to the project directory

```bash
  cd B-MODAL
```

Install dependencies

```bash
  yarn install
```

Compile the smart contract:

```bash
  npx hardhat compile
```

Start the development server

```bash
  npm run dev
```
Open [http://localhost:3000](http://localhost:3000)
